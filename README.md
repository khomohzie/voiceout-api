# shelter-api
This is the API and backend server for the Shelter mobile application.

### Error codes
1.
  - code: ACC_ERR_01
  - alias: acc_not_found
  - reason: account not found
  
2. 
  - code: ACC_ERR_02
  - alias: acc_not_verified
  - reason: account not verified

3.
  - code: ACC_ERR_03
  - alias: acc_not_admin
  - reason: account not admin
